# Exploiting New Properties of String Net Frequency for Efficient Computation


## Requirements
* `CMake 3.24+`
* `GCC` or `Clang`


## Building and running

Create the build directory:
```sh
$ mkdir build 
$ cd build
```

Detect the C++ compiler and create build files:
```sh
$ cmake ..
```

Run the build:
```sh
$ cmake --build .
```

Run the program:
```sh
$ ./main
```
